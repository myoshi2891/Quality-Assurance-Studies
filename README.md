# ソフトウェアテスト実施概要

本プロジェクトにおいて、システムの品質（正しく動くか・快適に使えるか）を担保するために実施するテスト項目の概要と目的を以下に定義します。

## 1. 機能テスト（Functional Testing）
システムが要件定義書および設計書通りに正しく機能するかを検証します。

* **単体テスト (Unit Testing)**
  * **概要:** 関数やクラスなど、プログラムの最小単位を対象としたテスト。
  * **目的・解決する課題:** 開発の初期段階でロジックの誤りを検知し、バグの修正コストを最小限に抑えます。
* **結合テスト (Integration Testing)**
  * **概要:** 複数のモジュールやデータベースなどを組み合わせた際の連携を検証するテスト。
  * **目的・解決する課題:** コンポーネント間のインターフェースの不整合や、データの受け渡しエラーを防ぎます。
* **E2Eテスト / 総合テスト (End-to-End Testing)**
  * **概要:** 実際のユーザー操作と同じシナリオで、システム全体を通しで検証するテスト。
  * **目的・解決する課題:** ビジネスロジック全体が正しく完了するかを確認し、ユーザー体験を損なう致命的な欠陥を防ぎます。
* **回帰テスト (Regression Testing)**
  * **概要:** コードの追加・修正が、既存の機能に悪影響（デグレ）を与えていないかを確認するテスト。
  * **目的・解決する課題:** 予期せぬサイドエフェクトによる既存機能の破壊を防ぎ、システムの安定性を継続的に担保します。
* **受入テスト (UAT: User Acceptance Testing)**
  * **概要:** 実際のユーザーや発注者が、要求を満たしているかを最終確認するテスト。
  * **目的・解決する課題:** 開発者目線ではなくユーザー目線での「本当に使えるシステム」になっているかを検証し、要件のズレを防ぎます。

## 2. 非機能テスト（Non-Functional Testing）
システムの性能、セキュリティ、操作性など、機能面以外の品質基準を満たしているかを検証します。

* **パフォーマンステスト (Performance / Load Testing)**
  * **概要:** 大量アクセスや高負荷時におけるシステムの応答速度や耐久性を検証するテスト。
  * **目的・解決する課題:** アクセス集中時のシステムダウンやレスポンス遅延による機会損失を防ぎます。
* **セキュリティテスト (Security Testing)**
  * **概要:** システムの脆弱性を洗い出し、悪意のある攻撃への耐性を検証するテスト。
  * **目的・解決する課題:** 個人情報の漏洩やデータの改ざんなど、致命的なセキュリティインシデントを未然に防ぎます。
* **互換性テスト (Compatibility Testing)**
  * **概要:** 異なるOS、ブラウザ、デバイス環境で正常に動作・表示されるかを検証するテスト。
  * **目的・解決する課題:** ユーザーの利用環境への依存による表示崩れや機能不全を防ぎます。
* **ユーザビリティ＆アクセシビリティテスト**
  * **概要:** 直感的な操作性や、多様なユーザー（身体的制約を持つ方を含む）にとっての使いやすさを検証するテスト。
  * **目的・解決する課題:** 特定のユーザーがシステムを利用できなくなる事態を防ぎ、誰もが快適に使えるUI/UXを実現します。

## 3. 開発ツール（Migration Tools）
既存の静的 HTML ページを Next.js 環境へ安全に移行するためのスクリプト群です。

* **CSS 抽出・変換 (`scripts/extract-css.mjs`)**
  * HTML からスタイルブロックを抽出し、プロジェクトのデザイントークン（Tailwind v4 変数）へ自動マッピングします。
* **HTML to JSX 変換 (`scripts/html-to-tsx.mjs`)**
  * HTML 構造を JSX に変換し、属性の置換（`class` → `className` 等）やプロジェクト標準クラスへのリネームを行います。

## qa_studies

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run 
```

This project was created using `bun init` in bun v1.3.5. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
